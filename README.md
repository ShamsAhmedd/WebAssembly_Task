# Quick Sort: JavaScript vs WebAssembly (WASM)

This task demonstrates the performance difference between **JavaScript** and **WebAssembly (WASM)** when executing a computationally intensive algorithm.

The same **Quick Sort** algorithm is implemented in:
- JavaScript
- C compiled to WebAssembly using **Emscripten**

The comparison focuses on **execution time**, especially for large arrays, to highlight where WebAssembly can provide performance benefits.
